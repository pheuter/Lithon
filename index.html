Lithon

Lithon is a toy compiler that translates a small subset of Scheme into Python. It implements the CodeTalker compiler compiler.

Let's say we want to compile the following Scheme into Python:

(define (factorial_rec n)
  (if (= n 0)
      1
      (* n (factorial_rec (- n 1)))))

(define (factorial_iter n)
  (reduce (lambda (x y) (* x y)) (xrange 1 (+ 1 n))))

(map factorial_rec (xrange 1 11))
(map factorial_iter (xrange 11 16))

We simply write the code to a file and pass it to lithon.py:

$ lithon.py tests/factorial.scm

We now have a compiled version of the code in a .py file under the same directory:

def factorial_rec(n):
  if n == 0:
    return 1
  else:
    return (n * factorial_rec(n-1))

def factorial_iter(n):
  return reduce((lambda x,y: x*y), xrange(1,n+1))

print map(factorial_rec, xrange(1,11))
print map(factorial_iter, xrange(11,16))
